Uses a 12 column base. Heavilly based on Flexbox.
  • Each group of columns must be enclosed in a block level element with a 'row' class.
  • At minimum each column must have the 'col' class, which will make them auto size based on the neighboring columns.
  • To specify a size, add a class of 'col-x', where x is the column size you'd like.
  • By default rows are left justified, but you can make them center and right justified by also adding the 'row-center' or 'row-right' classes, respectively.
  • Use the 'row-expand' class to make all columns space out to fill the row.
